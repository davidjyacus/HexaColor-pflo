About HexaColor

HexaColor is a solo project realized by David J. Yacus as part of the Scrimba Frontend Developer Career Path.

The main scope of the project is to showcase the use of APIs. Built with HTML, CSS, and JavaScript, it also demonstrates DOM manipulation, basic UI design, and responsive design.

HexaColor

Welcome to HexaColor, a simple tool for identifying and visualizing colors using their hexadecimal codes.

Simply choose a seed color, a color scheme mode, and click on the provided button to get the color scheme.
